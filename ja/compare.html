Windows向けの画面拡大ソフト

競合製品の詳細を2026年7月20日に確認しました

これまで拡大ソフトを使ったことがないなら、まずはWindowsにもともと内蔵されているものから始めてください。無料です。画面を読み上げてほしい、非常に高い拡大率が必要、または点字が必要という場合は、ZoomTextやSuperNovaのような総合的な支援技術スイートが必要で、Low Vision Zoomでは足りません。Low Vision Zoomは、1つのことだけを行う小さくて手ごろなツールです。つまり、カーソルを基準にした最大5倍までのなめらかな拡大で、設定するものは何もありません。

どれを選べばよいか

  • まだどの拡大ソフトも試していない — すでにお使いのPCに入っているWindows Magnifier（拡大鏡）を使ってください。Windows key + Plusを押します。費用はかからず、それだけで十分かもしれません。
  • 画面を読み上げてほしい、または点字を使う — ZoomText Magnifier/Reader、SuperNova、またはスクリーンリーダーが必要です。Low Vision Zoomには音声も点字対応も一切ありません。
  • 5倍を超える拡大が必要 — Low Vision Zoomは5倍で止まります。Windows Magnifier、ZoomText、SuperNovaはいずれも、はるかに高い倍率まで対応します。
  • カラーフィルター、コントラストの反転、フォーカスの強調表示が必要 — Windows Magnifierにはこの一部が無料で備わっており、ZoomTextとSuperNovaにはさらに多くの機能があります。Low Vision Zoomにはこれらは一切ありません。
  • 無料のサードパーティ製の選択肢が欲しい — Magnifixerは無料で、反転表示やカラーモードを備えた拡大ウィンドウを提供します。
  • おもに、作業しながら絶えず拡大・縮小したいのに、今それがぎくしゃくして煩わしいと感じている — これはLow Vision Zoomがまさに作られた場面です。カーソルのまわりをなめらかに拡大し、設定はなく、$9.99の一度きりのお支払いです。デモを見て、無料トライアルをお試しください。

Low Vision Zoomを買わない方がよい理由

私たちは、あなたが$9.99を、役に立たないものではなく役に立つものに使ってほしいと思っています。次のいずれかに当てはまる場合は、Low Vision Zoomを買わないでください:

  • 5倍を超える拡大が必要。これはアプリの動かせない上限であり、設定ではありません。5倍で足りないなら、この製品のどの部分もお役に立ちません。
  • 音声が必要。テキスト読み上げも、文書リーダーも、スクリーンリーダーもありません。いずれも$9.99の機能として予定されていません。
  • 快適に読むために色の反転やコントラストフィルターが必要。Windows Magnifierはこれを無料で行いますが、私たちはまったく行いません。
  • 高倍率で鮮明なテキストが必要。私たちは画面をそのまま拡大します。総合スイートはテキストを再描画して鮮明さを保ちます。高いズーム倍率では、その違いは大きく、重要です。
  • 提供元のサポート、トレーニング、または調達ルートが必要。私たちはメールアドレスを1つ持つだけの、とても小さな会社です。正式なサポートが求められる職場での配慮には向きません。
  • マウスやトラックパッドを使わない。この設計は、拡大したいものを指し示すことを中心にしています。キーボードショートカットはありますが、キーボードだけで操作する方は、ほかの製品の方が向いています。
  • Windows on ARM、または32ビットのPCを使っている。現時点では対応していません。

そもそも拡大ソフトは必要ないかもしれません

何かを買う前に、すでにお使いの機器にある無料の選択肢を試してみる価値があります。多くの方にとっては、それで十分です:

  • すべてを恒久的に大きくする: Windowsの設定 → システム → ディスプレイ → 拡大／縮小。これを150%や175%にすると、追加のソフトなしで、あらゆる場所の文字とアイコンがすべて大きくなります。
  • 文字だけを大きくする: Windowsの設定 → アクセシビリティ → テキストのサイズ。
  • Webページを拡大する: どのブラウザでもCtrl + Plus。ピクセルを拡大するのではなく文字を折り返し直すので、Webページを読むにはたいてい、どの拡大ソフトよりも読みやすくなります。
  • 内蔵の拡大鏡: Windows key + Plus。

拡大ソフトは、あらゆるアプリで一日を通して詳細に出入りする必要があるときに適したツールです。本当に必要だったのが常に大きい文字であるなら、それは適さないツールです。

Low Vision Zoomが向いている場面

Low Vision Zoomが生まれたのは、その開発者がある特定の1つのことを望んだからです。それは、一日中絶えず拡大したり元に戻したりしても、そのたびに画面上で見ていた場所を見失わないことでした。段階的にではなく連続的に拡大し、カーソルの真下の点を軸に回るので、見ているものはその場にとどまり、カーソルが画面の中央あたりを動いている間は表示を静止させたままにします。設定するものは何もなく、キー1つとスクロールホイール以外に覚えることもありません。

それが製品のすべてです。上の説明がまさにあなたの困りごとなら、7日間の無料トライアルが、あなたの目に合うかどうかを数分で教えてくれます。もし合わなければ、このページのほかの製品のどれかがあなたにより役立つはずで、私たちはむしろそちらをご案内します。なめらかな拡大がなぜ目で追いやすいのかについて、さらにお読みいただくこともできますし、よくある質問もご覧いただけます。
