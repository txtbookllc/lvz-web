Windowsでの画面拡大と、Low Vision Zoomがどう役立つか

拡大ソフトを選ぶ

Windows向けの最高の画面拡大ソフトはどれですか？

唯一の「最高」というものはありません。必要なものによって決まりますし、正しい最初の一歩は無料です。すべてのWindows PCにはすでにWindows Magnifierが付属しており（Windows key + Plusを押すと起動します）、多くの人にとってはそれで十分です。画面を音声で読み上げてほしい、非常に高い倍率が必要、色フィルターや点字が必要という場合は、ZoomTextやSuperNovaのような本格的な支援技術スイートが適しています。設定不要で、なめらかに、落ち着いてカーソルに追従するズームが欲しいだけなら、Low Vision Zoomはまさにその用途のために作られており、$9.99の買い切りです。拡大の好みは人それぞれなので、支払う前にどのオプションも試してみてください。正直な比較もご覧ください。

Windows Magnifierの代わりになるものはありますか？

はい。Microsoft純正のWindows Magnifierのほかに、Windows向けのサードパーティ製拡大鏡としては、音声と読み上げツールを備えた本格的なロービジョン向けスイートであるZoomTextとSuperNova、無料のMagnifixer、そしてなめらかでカーソルに追従するズームに絞った小さな$9.99のユーティリティであるLow Vision Zoomがあります。人がこうした代替を探すのは、より滑らかな使い心地、マウスポインターを中心にしたズーム、あるいは標準ツールにはない音声や色の機能が欲しいときです。どの代替が合うかは、そのうちどれが必要かによって決まります——比較ページで並べて確認できます。

ZoomTextより安価な代替品はありますか？

どのZoomTextの機能を使っているかによります。ZoomTextは、音声、ドキュメントリーダー、豊富な色・フォーカス強調機能、非常に高い倍率を備えた本格的なロービジョン向けスイートで、通常は年間ライセンスとして販売されています。そうした読み上げや強調機能を使っているなら、正直なところ、より安価なツールではそれらを代替できません。主に、カーソルに追従するなめらかな拡大だけが欲しいのであれば、Low Vision Zoomはその一点に絞って買い切り$9.99で提供します——ただし音声も読み上げも色フィルターもなく、倍率は5×までです。Windows Magnifierも無料で、多くの人が思っている以上に高機能です。

Low Vision Zoomはテキストを音声で読み上げますか？

いいえ。Low Vision Zoomは拡大機能のみです。テキスト読み上げも、ドキュメントリーダーも、スクリーンリーダーもありません。画面を音声で読み上げてほしい場合は、ZoomText Magnifier/Reader、音声版またはスクリーンリーダー版のSuperNova、あるいは無料であるWindows標準のNarratorをご検討ください。

Low Vision Zoomは、Windows標準の拡大鏡と何が違いますか？

Windows Magnifierは無料で標準搭載されており、高機能なので、多くの人はまずこれを試すべきです。Low Vision Zoomは機能の数ではなく、使い心地で差別化されています。ズームは段階的ではなく連続的で新しい倍率へなめらかに移行し、マウスカーソルの真下にある正確な一点を中心に拡大するため見ている場所がそのまま保たれ、カーソルが画面の中央あたりを動いている間は表示を静止させます。設定はなく、システムトレイに常駐します。またWindows Magnifierより劣る点もあります：音声がなく、色フィルターがなく、倍率は5×までです。なめらかな動きの理由について詳しくは、なめらかな拡大が追いやすい理由をご覧ください。

Low Vision Zoomの使い方

Windows 11でズームするにはどうすればいいですか？

Windowsには標準の拡大鏡があります：Windows keyを押しながらPlusを押すと拡大、Minusで縮小、Windows key + Escでオフになります。設定 → システム → ディスプレイ → 拡大率で画面全体を恒久的に大きくすることも、設定 → アクセシビリティ → テキストサイズで文字だけを大きくすることもできます。Low Vision Zoomをインストールすると、Ctrlを押しながらマウスホイールを回すとWindowsのどこでもなめらかにズームでき、マウスホイールがない場合はCtrl + Alt + Plus / Minusを使えます。

マウスホイールでズームできる拡大ソフトはありますか？

はい。Low Vision ZoomはCtrl + マウスホイールでズームします：Ctrlを押しながらホイールを前に回すと拡大、後ろに回すと縮小します。ズームは比例的で、ホイールを回した分だけ、回した速さで進み、ポインターの下の一点を中心に保ったままなめらかに新しい倍率へ移行します。Ctrl + Shift + スクロールダウンで瞬時に等倍へ戻ります。これがこのアプリの基本の使い方です。

Low Vision Zoomは複数モニターで使えますか？

はい。Windowsのデスクトップ全体を1つの連続した空間として拡大し、それぞれの画面を別々に扱うのではなく、モニターをまたいでカーソルに追従します。これはWindows標準の拡大鏡の全画面モードと同じ考え方です。

マウスホイールのないノートPCではどうやってズームしますか？

キーボードを使います。Ctrl + Altを押しながらPlusを押すと拡大、Minusを押すと縮小します。押し続けるとズームし続けます。Ctrl + Alt + 0で等倍にリセットされます。マウスホイールも外付けマウスも不要なので、トラックパッドしかないノートPCでも使えます。

ゲームや全画面動画でも動作しますか？

ほぼどこでも動作します。Low Vision Zoomは、Windowsデスクトップ全体を拡大するので、ブラウザ、メール、ドキュメント、ウィンドウ内で再生される動画、Windowsのシェル自体にも適用されます。例外は、通常のデスクトップを経由せず画面全体を排他的に制御するゲームやアプリで、その場合は拡大されないことがあります。そうしたゲームをウィンドウモードやボーダーレスウィンドウモードで実行すると、拡大機能が復活します。

Low Vision Zoomは画面をどれくらい拡大できますか？

最大5×まで、0.5刻みで拡大できます。これは、小さく快適なズームツールとしてあえて設けた上限で、Windows標準の拡大鏡（最大16×）やZoomTextのような本格スイート（最大36×）よりも低い値です。5×を超える倍率が必要な場合、Low Vision Zoomは適した選択肢ではありません——比較ページでより高倍率の製品を紹介しています。
