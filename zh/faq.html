Windows 上的屏幕放大，以及 Low Vision Zoom 的定位

选择放大镜

Windows 上最好的屏幕放大镜是什么？

没有绝对的最好——这取决于您的具体需求，而正确的第一步是免费的。每台 Windows 电脑都已经内置了 Windows Magnifier（按 Windows key + Plus 即可启动）；对许多人来说，这已经够用了。如果您需要朗读屏幕内容、非常高的放大倍数、颜色滤镜或盲文支持，那么应该选择 ZoomText 或 SuperNova 这类完整的辅助技术套件。如果您想要的是平滑、平静、跟随光标的缩放，且无需配置任何设置，Low Vision Zoom 正是为这种具体需求而打造的，只需一次性支付 $9.99。由于放大镜的选择因人而异，请在付费前先试用任意方案。参见我们的诚实对比。

有没有 Windows Magnifier 的替代方案？

有。除了微软内置的 Windows Magnifier 之外，Windows 上的第三方放大镜还包括 ZoomText 和 SuperNova（带有语音和阅读工具的完整低视力套件）、免费的 Magnifixer，以及 Low Vision Zoom——一款专注于平滑、以光标为锚点的缩放的 $9.99 小工具。人们通常在想要更流畅的手感、以鼠标指针为中心的缩放，或内置工具没有的语音和颜色功能时，才会寻找替代品。哪种替代方案更合适，取决于您具体需要以上哪一项——对比页面将它们并排列出。

有没有比 ZoomText 更便宜的替代品？

这取决于您依赖 ZoomText 的哪些功能。ZoomText 是一款完整的低视力套件，带有语音、文档阅读器、丰富的颜色与焦点增强功能，以及非常高的放大倍数，通常以年度许可证的形式出售。如果您使用的正是这些阅读和增强功能，坦白说，更便宜的工具无法替代它们。如果您主要想要的是跟随光标的平滑放大，Low Vision Zoom 只专注于这一件事，一次性支付 $9.99 即可——但它没有语音、没有阅读功能、没有颜色滤镜，且最高只能放大到 5×。Windows Magnifier 同样免费，功能也比许多人以为的更强大。

Low Vision Zoom 会朗读文本吗？

不会。Low Vision Zoom 只是一款放大工具。它没有文本转语音功能，没有文档阅读器，也没有屏幕阅读器。如果您需要朗读屏幕内容，请考虑 ZoomText Magnifier/Reader、带语音或屏幕阅读器版本的 SuperNova，或者 Windows 内置且免费的 Narrator。

Low Vision Zoom 与内置的 Windows Magnifier 有什么不同？

Windows Magnifier 是免费的、内置的，而且功能强大，许多人应该先试试它。Low Vision Zoom 的不同之处在于手感，而非功能数量。它的缩放是连续的，会平滑过渡到新的倍数而不是跳跃式变化；它围绕鼠标光标正下方的确切位置进行缩放，让您注视的内容保持不动；并且当光标在屏幕中部移动时，画面会保持静止。它没有任何设置，常驻在系统托盘中。它在某些方面也比 Windows Magnifier 更受限：没有语音、没有颜色滤镜，且最高只能放大到 5×。关于平滑动画背后的原理，详见为什么平滑放大更容易跟随。

使用 Low Vision Zoom

如何在 Windows 11 上放大屏幕？

Windows 自带一个放大镜：按住 Windows key 并按 Plus 放大，Minus 缩小，Windows key + Esc 关闭。您也可以在设置 → 系统 → 显示 → 缩放中永久放大所有内容，或在设置 → 无障碍 → 文本大小中只放大文字。安装 Low Vision Zoom 后，按住 Ctrl 并滚动鼠标滚轮即可在 Windows 中任意位置平滑缩放；如果没有鼠标滚轮，可以使用 Ctrl + Alt + Plus / Minus。

有没有可以用鼠标滚轮缩放的放大镜？

有。Low Vision Zoom 使用 Ctrl + 鼠标滚轮进行缩放：按住 Ctrl 并向前滚动放大，向后滚动缩小。缩放是成比例的，您滚动多远、多快，画面就相应地缩放多少，并且会平滑过渡到新倍数，同时始终保持以指针下方的点为中心。Ctrl + Shift + 向下滚动可立即恢复到正常大小。这是应用程序设计的主要使用方式。

Low Vision Zoom 支持多台显示器吗？

支持。它将整个 Windows 桌面作为一个连续的空间进行放大，并在您的光标从一个显示器移动到另一个显示器时持续跟随，而不是把每块屏幕当作独立区域处理。这与内置 Windows Magnifier 全屏模式的方式相同。

在没有鼠标滚轮的笔记本电脑上如何缩放？

使用键盘。按住 Ctrl + Alt 并按 Plus 放大，或按 Minus 缩小；按住不放即可持续缩放。Ctrl + Alt + 0 可重置为正常大小。无需鼠标滚轮或外接鼠标，因此该应用在只有触控板的笔记本电脑上也能正常工作。

它在游戏和全屏视频中也能使用吗？

几乎在任何地方都能使用。Low Vision Zoom 会放大整个 Windows 桌面，因此适用于浏览器、邮件、文档、窗口中播放的视频，以及 Windows 外壳本身。唯一的例外是那些独占整个屏幕、绕过常规桌面的游戏或应用；这类内容可能无法被放大。以窗口模式或无边框窗口模式运行此类游戏即可恢复放大功能。

Low Vision Zoom 最高能放大多少倍？

最高 5×，以半步为增量递增。这是为一款小巧、舒适的缩放工具刻意设定的限制，低于内置 Windows Magnifier（最高 16×）或 ZoomText 等完整套件（最高 36×）。如果您需要超过 5× 的放大倍数，Low Vision Zoom 不适合您——对比页面会为您指出放大倍数更高的方案。
