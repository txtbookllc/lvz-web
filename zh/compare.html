适用于 Windows 的屏幕放大镜

竞品信息核实于 2026 年 7 月 20 日

如果您从未用过放大镜，请先试试 Windows 中已经内置的那一款。它是免费的。如果您需要将电脑内容朗读出来，或需要非常高的放大倍数，或需要盲文，那么您需要一套完整的辅助技术套件，例如 ZoomText 或 SuperNova，此时 Low Vision Zoom 将无法满足您的需求。Low Vision Zoom 是一款小巧、经济的工具，只做一件事：平滑、锚定光标的放大，最高 5×，没有任何需要配置的地方。

您应该选择哪一款？

  • 您还没有试用过任何放大镜——请使用您电脑上已有的 Windows Magnifier。按 Windows key + Plus。它不花一分钱，而且可能已经满足您的全部需求。
  • 您需要将屏幕内容朗读出来，或者您使用盲文——您需要 ZoomText Magnifier/Reader、SuperNova 或一款屏幕阅读器。Low Vision Zoom 完全没有语音，也完全不支持盲文。
  • 您需要超过 5× 的放大倍数——Low Vision Zoom 最高只到 5×。Windows Magnifier、ZoomText 和 SuperNova 都能达到高得多的倍数。
  • 您需要颜色滤镜、对比度反转或焦点高亮——Windows Magnifier 免费提供其中一部分功能；ZoomText 和 SuperNova 提供的更多。Low Vision Zoom 完全没有这些功能。
  • 您想要一个免费的第三方选择——Magnifixer 是免费的，提供一个带有反色显示和颜色模式的放大镜窗口。
  • 您主要想在工作时不断地放大和缩小，而目前觉得这个过程跳动或繁琐——这正是 Low Vision Zoom 专为之打造的具体场景。它围绕您的光标平滑放大，没有任何设置，一次性收费 $9.99。观看演示并试用免费版。

不购买 Low Vision Zoom 的理由

我们宁愿您把 $9.99 花在对您有帮助的东西上，而不是花在没有帮助的东西上。如果以下任何一条适用于您，请不要购买 Low Vision Zoom：

  • 您需要超过 5× 的放大倍数。这是应用中的硬性上限，而不是一项设置。如果 5× 对您来说不够，本产品的任何部分都帮不上忙。
  • 您需要语音。没有文字转语音，没有文档阅读器，也没有屏幕阅读器。这些都不在 $9.99 功能的计划之内。
  • 您需要颜色反转或对比度滤镜才能舒适地阅读。Windows Magnifier 免费提供此功能；我们完全不提供。
  • 您需要在高倍放大下保持清晰的文本。我们按屏幕原样放大；那些套件会重新渲染文本以保持清晰。在高缩放倍数下，这个差异很大，而且很重要。
  • 您需要厂商支持、培训或采购渠道。我们是一家非常小的公司，只有一个电子邮件地址。对于需要正式支持的工作场所无障碍方案来说，这并不合适。
  • 您不使用鼠标或触控板。该设计的核心是指向您想放大的内容。虽然有键盘快捷键，但仅使用键盘的用户在别处会得到更好的服务。
  • 您使用的是 Windows on ARM 或 32 位电脑。目前不支持。

您也许根本不需要放大镜

在购买任何东西之前，值得先试试您电脑上已有的免费选项。对许多人来说，它们已经足够：

  • 永久放大所有内容：Windows 设置 → 系统 → 显示 → 缩放。将其设为 150% 或 175% 会在各处放大所有文字和图标，无需任何额外软件。
  • 仅放大文字：Windows 设置 → 辅助功能 → 文本大小。
  • 缩放网页：在任何浏览器中按 Ctrl + Plus，它会重新排版文字，而不是放大像素——就阅读网页而言，通常比任何放大镜都更易读。
  • 内置放大镜：Windows key + Plus。

当您需要一整天在所有应用中不断进出细节时，放大镜是合适的工具。当您真正需要的只是始终更大的文字时，它就是错误的工具。

Low Vision Zoom 适合的场景

Low Vision Zoom 之所以存在，是因为它的开发者想要一件具体的事：一整天不断地放大再缩小，而每次都不丢失自己在屏幕上的位置。它连续放大而不是分档放大，围绕光标下方的点旋转，让您正在看的东西保持不动，并在您的光标在屏幕中部游走时保持视图静止。除了一个按键和滚轮之外，没有任何需要设置或学习的东西。
