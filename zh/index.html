为什么更好：它从不让您重新寻找刚才的位置。

大多数放大镜以突兀的跳跃方式移动，每一次跳跃都迫使您的眼睛停下来，重新寻找刚才看到的地方。Low Vision Zoom 的移动是平滑而连续的。

眼睛能够跟随的动画

放大镜瞬间跳动时，您的眼睛不得不追上去，重新寻找刚才正在阅读的内容。而在这里，每一次变化都以每秒 60 帧的动画呈现，您的眼睛可以顺着画面一起滑行，而不是去追赶它。

您的锚点从不移动

缩放围绕光标正下方的那个点进行。这个点保持不动，周围的一切逐渐放大。您只需指向重要的内容，就选定了自己的锚点。

同一画面，而不是新屏幕

每一帧都只比上一帧前进一小步，因此画面始终像同一页面在平滑地放大，而不是新屏幕替换旧屏幕。没有任何东西消失，也就没有什么需要重新寻找。

节奏由您掌控

缩放与滚动成正比：滚动多远、多快，就缩放多远、多快，而且您可以在动作进行中随时停下或反向。由您自己发起的移动，是您的眼睛能够预测的移动。

阅读时画面静止

当光标在屏幕中部移动时，画面完全保持静止；只有当光标靠近边缘时，画面才会平滑滑动。您阅读的内容不会一直在漂移。

扎实的 Windows 工程

基于 Windows 自身的 Magnification API 构建，并配备自己的 60 fps 动画引擎。安装程序经过签名，放大状态下点击也能准确落位，一切都在本地运行。任何数据都不会离开您的电脑。

专为 Windows 11（64 位，x64）打造 · 独立完整安装程序，无需再下载其他组件 · ARM 支持即将推出。

小知识：视觉研究者称之为变化盲视（change blindness）：当画面在视觉中断期间发生变化时，人很容易错过移动的部分，丢失正在跟踪的目标。跳跃式的放大镜整天都在重复制造这种中断，而平滑滑动的放大镜从不给您的眼睛留下迷失位置的空隙。

继续安装的方法：

  1. 在蓝色界面上点击“更多信息”。
  2. 确认发布者一行包含 txtbook LLC。
  3. 点击随后出现的“仍要运行”按钮，安装程序就会照常打开。
