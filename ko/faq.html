Windows 화면 확대와 Low Vision Zoom이 어떻게 어울리는지

확대 프로그램 선택하기

Windows용 최고의 화면 확대 프로그램은 무엇인가요?

딱 하나의 정답은 없습니다 — 무엇이 필요한지에 따라 다르며, 가장 먼저 해볼 만한 방법은 무료입니다. 모든 Windows PC에는 이미 Windows Magnifier가 기본으로 포함되어 있어(Windows key + Plus 키를 눌러 시작) 많은 분들에게는 이것으로 충분합니다. 화면을 소리 내어 읽어주는 기능, 매우 높은 배율, 색상 필터, 점자가 필요하다면 ZoomText나 SuperNova 같은 종합 보조공학 제품군이 적합한 범주입니다. 설정할 것 없이 작업 중 커서를 부드럽고 차분하게 따라오는 확대를 원하신다면, Low Vision Zoom이 바로 그런 경우를 위해 만들어졌으며 $9.99에 한 번만 결제하면 됩니다. 확대 방식은 사람마다 다르게 느껴지므로, 결제 전에 어떤 옵션이든 먼저 사용해 보세요. 저희의 공정한 비교를 참고하세요.

Windows Magnifier의 대안이 있나요?

네. Microsoft가 기본 제공하는 Windows Magnifier 외에도 Windows용 타사 확대 프로그램으로는 ZoomText와 SuperNova(음성과 읽기 도구를 갖춘 종합 저시력 제품군), 무료인 Magnifixer, 그리고 부드럽고 커서에 고정된 확대에 집중한 $9.99짜리 소형 유틸리티 Low Vision Zoom이 있습니다. 사람들은 보통 더 부드러운 사용감, 마우스 포인터 중심의 확대, 또는 기본 도구에는 없는 음성·색상 기능을 원할 때 대안을 찾습니다. 어떤 대안이 맞는지는 이 중 무엇이 필요한지에 따라 달라집니다 — 비교 페이지에서 나란히 정리해 두었습니다.

ZoomText보다 저렴한 대안이 있나요?

어떤 ZoomText 기능을 사용하시는지에 따라 다릅니다. ZoomText는 음성, 문서 읽기, 폭넓은 색상·초점 보강 기능, 매우 높은 배율을 갖춘 종합 저시력 제품군으로, 보통 연간 라이선스로 판매됩니다. 그런 읽기·보강 기능을 사용하신다면, 솔직히 말해 더 저렴한 도구로는 이를 대체할 수 없습니다. 주로 커서를 따라오는 부드러운 확대만 원하신다면, Low Vision Zoom이 바로 그 한 가지를 일회성 $9.99 결제로 제공합니다 — 다만 음성, 읽기, 색상 필터가 없고 5×에서 멈춥니다. Windows Magnifier도 무료이며 많은 분들이 생각하는 것보다 더 강력합니다.

Low Vision Zoom이 텍스트를 소리 내어 읽어주나요?

아니요. Low Vision Zoom은 확대 기능만 제공합니다. 텍스트 음성 변환, 문서 읽기, 스크린 리더 기능이 없습니다. 화면을 소리 내어 읽어주는 기능이 필요하시다면 ZoomText Magnifier/Reader, 음성 또는 스크린 리더 버전의 SuperNova, 또는 무료인 Windows 기본 제공 Narrator를 확인해 보세요.

Low Vision Zoom은 Windows에 기본 내장된 Windows Magnifier와 무엇이 다른가요?

Windows Magnifier는 무료이고 기본 제공되며 성능도 좋아서 많은 분들이 먼저 사용해 볼 만합니다. Low Vision Zoom은 기능 개수가 아니라 사용감에서 차이가 납니다. 확대가 연속적이며 단계적으로 끊기지 않고 새로운 배율로 부드럽게 이어지고, 마우스 커서 바로 아래 지점을 중심으로 확대되어 보고 있던 곳이 그대로 유지되며, 커서가 화면 가운데를 오가는 동안에는 화면이 정지해 있습니다. 별도 설정이 없고 시스템 트레이에 상주합니다. 반면 Windows Magnifier보다 제한적인 부분도 있습니다: 음성 기능이 없고, 색상 필터가 없으며, 5×에서 멈춥니다. 부드러운 움직임에 관한 근거는 부드러운 확대가 따라가기 쉬운 이유에서 더 자세히 다룹니다.

Low Vision Zoom 사용하기

Windows 11에서 화면을 확대하려면 어떻게 하나요?

Windows에는 기본 제공 확대 프로그램이 있습니다: Windows key를 누른 채 Plus를 누르면 확대되고, Minus를 누르면 축소되며, Windows key + Esc를 누르면 꺼집니다. 설정 → 시스템 → 디스플레이 → 배율에서 화면 전체를 영구적으로 키우거나, 설정 → 접근성 → 텍스트 크기에서 텍스트만 키울 수도 있습니다. Low Vision Zoom을 설치하면 Ctrl 키를 누른 채 마우스 휠을 굴려 Windows 어디서나 부드럽게 확대할 수 있으며, 마우스 휠이 없다면 Ctrl + Alt + 플러스 / 마이너스를 사용하세요.

마우스 휠로 확대하는 확대 프로그램이 있나요?

네. Low Vision Zoom은 Ctrl + 마우스 휠로 확대합니다: Ctrl을 누른 채 휠을 앞으로 굴리면 확대되고, 뒤로 굴리면 축소됩니다. 확대는 비례적이어서 휠을 돌린 만큼, 돌린 속도로 정확히 움직이며, 포인터 아래 지점에 고정된 채 새로운 배율로 부드럽게 이어집니다. Ctrl + Shift + 스크롤 아래로를 하면 즉시 원래대로 돌아갑니다. 이것이 앱을 사용하도록 설계된 주된 방식입니다.

Low Vision Zoom은 여러 대의 모니터에서도 작동하나요?

네. Windows 바탕 화면 전체를 하나의 연속된 공간으로 확대하며, 각 화면을 따로 다루지 않고 한 모니터에서 다음 모니터로 커서를 따라갑니다. 이는 Windows 기본 제공 Windows Magnifier의 전체 화면 모드와 같은 방식입니다.

마우스 휠이 없는 노트북에서는 어떻게 확대하나요?

키보드를 사용하세요. Ctrl + Alt를 누른 채 플러스를 눌러 확대하거나 마이너스를 눌러 축소합니다. 키를 계속 누르고 있으면 계속 확대됩니다. Ctrl + Alt + 0을 누르면 원래대로 돌아갑니다. 마우스 휠이나 외장 마우스가 필요 없으므로 트랙패드만 있는 노트북에서도 작동합니다.

게임이나 전체 화면 동영상에서도 작동하나요?

거의 모든 곳에서 작동합니다. Low Vision Zoom은 Windows 바탕 화면 전체를 확대하므로 브라우저, 이메일, 문서, 창 안에서 재생되는 동영상, Windows 셸 자체에도 적용됩니다. 예외는 일반 바탕 화면을 우회하여 전체 화면을 독점적으로 제어하는 게임이나 앱인데, 이런 콘텐츠는 확대되지 않을 수 있습니다. 그런 게임을 창 모드나 테두리 없는 창 모드로 실행하면 확대 기능이 복원됩니다.

Low Vision Zoom은 화면을 얼마나 확대할 수 있나요?

최대 5×까지, 0.5배 단위로 조절됩니다. 작고 편안한 확대 도구로 남기 위한 의도적인 제한이며, Windows 기본 제공 Windows Magnifier(최대 16×)나 ZoomText 같은 종합 제품군(최대 36×)보다 낮습니다. 5× 이상이 필요하시다면 Low Vision Zoom은 맞지 않는 도구입니다 — 비교 페이지에서 더 높은 배율을 제공하는 제품을 안내해 드립니다.

구매, 체험, 신뢰

Low Vision Zoom의 가격은 얼마인가요?

이후 모든 업데이트가 포함된 $9.99 일회성 라이선스입니다. 구독도, 계정도, 광고도 없습니다. 먼저 7일간 모든 기능을 무료로 사용해 볼 수 있으며, 구매 후에는 30일 환불 보장이 제공됩니다. 가격 페이지를 참고하세요.

무료 체험이 끝나면 어떻게 되나요?

앱은 계속 작동합니다. 7일 체험이 끝나면 확대 기능이 꺼지는 것이 아니라 단순히 제한될 뿐이므로, 화면은 항상 탐색하고 라이선스를 구매하기에 충분할 만큼 읽을 수 있는 상태로 유지됩니다. 언제든 라이선스 키를 입력하면 재설치 없이 즉시 전체 확대가 복원됩니다.

Low Vision Zoom은 데이터를 수집하나요? 오프라인에서도 작동하나요?

이 앱은 개인정보, 분석 데이터, 원격 측정(텔레메트리)을 전혀 수집하지 않으며 완전히 오프라인으로 작동합니다. 화면 내용을 캡처하거나 전송하지 않습니다. 라이선스 키는 계정이나 서버 통신 없이 기기에서 확인됩니다. 인터넷에 연결되는 유일한 순간은 트레이 메뉴에서 업데이트 확인을 선택할 때뿐입니다. 자세한 내용은 개인정보 처리방침 (영문)을 참고하세요.

Low Vision Zoom을 설치할 때 Windows가 경고를 표시하나요?

직접 다운로드를 사용하신다면 그럴 수 있습니다. 설치 프로그램을 처음 열면 Windows가 파란색 “Windows의 PC 보호(Windows protected your PC)” SmartScreen 메시지를 표시할 수 있습니다. 이는 새로운 앱에서 흔한 일이며 안전합니다. 설치 프로그램은 txtbook LLC가 디지털 서명했습니다. 추가 정보를 클릭해 게시자가 txtbook LLC인지 확인한 다음 실행을 클릭하세요. 이 화면을 아예 건너뛰고 싶다면 Microsoft Store에서 설치하세요 — 스토어를 통한 설치에서는 이 경고가 표시되지 않습니다.

시스템 요구 사항은 무엇인가요?

Low Vision Zoom은 Windows 10 및 Windows 11, 64비트(x64)에서 실행됩니다. 설치 프로그램은 자체 완결형이라 추가로 다운로드할 것이 없습니다. Windows on ARM 및 32비트 PC는 현재 지원되지 않습니다.
