How to use it

One key and your mouse wheel. That's the whole interface.

Ctrl + Scroll ↑

Zoom in — magnify smoothly toward your cursor.

Ctrl + Scroll ↓

Zoom out — ease back out with the same gentle motion.

Ctrl + Shift + Scroll ↓

Reset — snap instantly to normal (1×) when you need it.

Move the mouse

Pan — move toward a screen edge and the view glides to follow.

Getting started

  1. Install and launch. A magnifying-glass icon appears in your system tray, and a short About window shows the controls.
  2. Hold Ctrl and scroll to zoom in any window — it works system-wide.
  3. Right-click the tray icon to turn zoom on or off, start it with Windows, or exit.
  4. Set and forget. With "start with Windows" on, it comes up quietly in the tray, ready whenever you need it.

How is this different from the built-in Windows Magnifier?

Windows includes a capable magnifier, and many people should try it first. Low Vision Zoom focuses on a smoother, calmer feel: gentle animated zoom (no snapping), zoom centered on your cursor, steady dead-zone panning, and a stay-out-of-the-way tray interface for quick, frequent adjustments.

Will Windows warn me when I install it?

The installer is Authenticode-signed, which clears SmartScreen "unknown publisher" warnings as the signing reputation builds. It installs per-machine to Program Files, which is also what lets clicks land correctly while zoomed.
