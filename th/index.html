ชมการทำงานของ Low Vision Zoom

แค่ปุ่มเดียวกับล้อเมาส์ ก็ใช้งานได้ครบ

ใช้งานได้ทันทีหลังติดตั้ง ไม่ต้องตั้งค่าใดๆ

แสดงท่าทางการซูมตามประเภทอุปกรณ์ของคุณ

กด Ctrl ค้างไว้แล้วเลื่อนล้อเมาส์ — เลื่อนไปข้างหน้าเพื่อซูมเข้า เลื่อนกลับเพื่อซูมออก

กด Ctrl + Alt ค้างไว้แล้วกด + หรือ − — ไม่ต้องใช้ล้อเมาส์

Ctrl + Scroll ↑

ซูมเข้า — ขยายภาพอย่างนุ่มนวลเข้าหาเคอร์เซอร์ของคุณ

Ctrl + Scroll ↓

ซูมออก — ย่อกลับออกมาด้วยการเคลื่อนไหวที่นุ่มนวลแบบเดียวกัน

Ctrl + Alt + Plus / Minus

ซูมจากแป้นพิมพ์ — กดค้างไว้เพื่อซูมต่อเนื่อง ไม่ต้องใช้ล้อเมาส์

Ctrl + Shift + Scroll ↓

รีเซ็ต — กลับสู่ขนาดปกติ (1×) ทันทีเมื่อต้องการ

Ctrl + Alt + 0

รีเซ็ต — กลับสู่ขนาดปกติ (1×) ทันที

Move the mouse

Move the cursor

เลื่อนมุมมอง — เลื่อนเข้าใกล้ขอบจอ แล้วมุมมองจะไหลตามไปอย่างนุ่มนวล

ทำไมถึงดีกว่า: ไม่มีวันทำให้คุณต้องหาตำแหน่งเดิมใหม่

โปรแกรมขยายหน้าจอส่วนใหญ่เคลื่อนแบบกระตุกทันที และทุกครั้งที่กระตุก สายตาของคุณต้องหยุดและหาตำแหน่งเดิมที่เคยมองอยู่ Low Vision Zoom เคลื่อนไหวอย่างนุ่มนวลและต่อเนื่อง

การเคลื่อนไหวที่สายตาติดตามได้สบาย

เมื่อโปรแกรมขยายหน้าจอกระตุกเปลี่ยนทันที สายตาของคุณต้องกระโดดตามและหาจุดที่กำลังอ่านใหม่ แต่ที่นี่ทุกการเปลี่ยนแปลงเป็นแอนิเมชันที่ 60 เฟรมต่อวินาที ทำให้สายตาของคุณไหลตามไปอย่างนุ่มนวลแทนที่จะต้องไล่ตาม

จุดยึดของคุณไม่เคยขยับ

การซูมจะหมุนรอบจุดที่อยู่ใต้เคอร์เซอร์ของคุณพอดี จุดนั้นจะอยู่นิ่งในขณะที่ทุกอย่างขยายใหญ่ขึ้นรอบๆ คุณเลือกจุดยึดได้ง่ายๆ เพียงแค่ชี้ไปที่สิ่งที่สำคัญ

ฉากเดียวกัน ไม่ใช่หน้าจอใหม่

แต่ละเฟรมเป็นการเปลี่ยนแปลงเล็กน้อยจากเฟรมก่อนหน้า มุมมองจึงดูเหมือนหน้าเดิมที่ค่อยๆ ขยายใหญ่ขึ้นอย่างนุ่มนวล ไม่ใช่หน้าจอใหม่มาแทนที่หน้าเก่า ไม่มีอะไรหายไป จึงไม่มีอะไรต้องหาใหม่

คุณกำหนดจังหวะเอง

การซูมเป็นสัดส่วนโดยตรง: ซูมได้ระยะและความเร็วตามที่คุณเลื่อนล้อเมาส์พอดี และคุณสามารถหยุดหรือย้อนกลับได้ระหว่างการเคลื่อนไหว การเคลื่อนไหวที่คุณเป็นผู้เริ่มเองคือการเคลื่อนไหวที่สายตาของคุณคาดเดาได้

นิ่งสนิทขณะที่คุณอ่าน

มุมมองจะนิ่งสนิทตราบใดที่เคอร์เซอร์ของคุณยังอยู่กลางจอ และจะเลื่อนไหลก็ต่อเมื่อคุณเข้าใกล้ขอบจอเท่านั้น ไม่มีการเลื่อนไหลรบกวนขณะที่คุณกำลังอ่าน

วิศวกรรมที่แข็งแกร่งบน Windows

สร้างขึ้นบน Magnification API ของ Windows เอง พร้อมเอนจินแอนิเมชัน 60 เฟรมต่อวินาทีในตัว ตัวติดตั้งมีลายเซ็นดิจิทัล การคลิกทำงานถูกต้องแม้ขณะซูมอยู่ และทุกอย่างทำงานในเครื่องของคุณเท่านั้น ไม่มีข้อมูลใดออกจากพีซีของคุณ

ออกแบบมาสำหรับ Windows 10 และ 11, 64 บิต (x64) · ตัวติดตั้งครบในตัว ไม่ต้องดาวน์โหลดอย่างอื่นเพิ่ม · ยังไม่รองรับ Windows on ARM ในขณะนี้

ข้อมูลที่ควรรู้: นักวิจัยด้านการมองเห็นเรียกปรากฏการณ์นี้ว่า change blindness (ภาวะมองไม่เห็นการเปลี่ยนแปลง): เมื่อมุมมองเปลี่ยนไปในช่วงที่สายตาถูกขัดจังหวะ เป็นเรื่องง่ายที่จะพลาดสิ่งที่เคลื่อนไหวและหลงตำแหน่งที่กำลังติดตามอยู่ โปรแกรมขยายหน้าจอที่กระตุกเปลี่ยนจะสร้างการขัดจังหวะแบบนี้ตลอดทั้งวัน ส่วนโปรแกรมที่เคลื่อนไหวนุ่มนวลจะไม่เปิดช่องให้สายตาของคุณหลงตำแหน่งเลย

คำถามที่พบบ่อย

ราคาเท่าไหร่?

Low Vision Zoom ให้ทดลองใช้ฟรี 7 วันโดยปลดล็อกทุกฟีเจอร์ หลังจากนั้น ใบอนุญาตแบบจ่ายครั้งเดียว $9.99 จะทำให้ใช้งานต่อได้ — ไม่มีค่าสมัครสมาชิก และรวมการอัปเดตในอนาคตทั้งหมด ดูรายละเอียดที่หน้าราคา

เกิดอะไรขึ้นเมื่อช่วงทดลองใช้สิ้นสุด?

แอปยังคงทำงานต่อไป — เพียงแต่การขยายภาพจะถูกจำกัดจนกว่าคุณจะกรอกคีย์ใบอนุญาต ทำให้หน้าจอของคุณยังคงอ่านได้เพียงพอสำหรับการใช้งานและการซื้อ กรอกคีย์เมื่อใดก็ได้เพื่อคืนการซูมเต็มรูปแบบทันที ไม่ต้องติดตั้งใหม่

Windows จะเตือนตอนที่ฉันติดตั้งไหม?

ครั้งแรกที่คุณเปิดตัวติดตั้ง Windows อาจแสดงข้อความเต็มจอสีน้ำเงินว่า “Windows protected your PC” (Windows ปกป้อง PC ของคุณ) นี่เป็นเรื่องปกติและแอปนี้ปลอดภัย ที่เป็นเช่นนี้เพราะ Low Vision Zoom เป็นแอปใหม่ล่าสุด — Windows ยังไม่เคยเห็นมันบ่อยพอที่จะจดจำได้ ตัวติดตั้งมีลายเซ็นดิจิทัลจาก txtbook LLC บริษัทของเรา หากต้องการข้ามหน้าจอนี้ไปเลย ให้ติดตั้งจาก Microsoft Store แทน — การติดตั้งผ่าน Store จะไม่แสดงคำเตือน SmartScreen

วิธีดำเนินการต่อ:

  1. คลิก More info (ข้อมูลเพิ่มเติม) บนหน้าจอสีน้ำเงิน
  2. ตรวจสอบว่าบรรทัดผู้เผยแพร่มีชื่อ txtbook LLC
  3. คลิกปุ่ม Run anyway (เรียกใช้ต่อไป) ที่ปรากฏขึ้น แล้วตัวติดตั้งจะเปิดตามปกติ

คำเตือนนี้จะหายไปเองในสัปดาห์ถัดๆ ไปเมื่อมีคนติดตั้งแอปมากขึ้น และคุณต้องทำขั้นตอนนี้เพียงครั้งเดียว แอปติดตั้งแบบต่อเครื่องลงใน Program Files ซึ่งเป็นสิ่งที่ทำให้การคลิกทำงานถูกต้องขณะซูมอยู่ด้วย
