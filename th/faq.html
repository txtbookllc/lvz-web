การขยายหน้าจอบน Windows และตำแหน่งของ Low Vision Zoom

การเลือกโปรแกรมขยายหน้าจอ

โปรแกรมขยายหน้าจอที่ดีที่สุดสำหรับ Windows คืออะไร?

ไม่มีตัวเลือกที่ดีที่สุดเพียงหนึ่งเดียว — ทั้งนี้ขึ้นอยู่กับสิ่งที่คุณต้องการ และขั้นตอนแรกที่ถูกต้องคือการทดลองใช้แบบไม่เสียค่าใช้จ่าย พีซี Windows ทุกเครื่องมี Windows Magnifier ติดตั้งมาให้อยู่แล้ว (กด Windows key + Plus เพื่อเริ่มใช้งาน) สำหรับหลายคนเพียงเท่านี้ก็เพียงพอ หากคุณต้องการให้อ่านหน้าจอออกเสียง ขยายภาพในระดับสูงมาก ตัวกรองสี หรืออักษรเบรลล์ ชุดโปรแกรมเทคโนโลยีสิ่งอำนวยความสะดวกแบบครบวงจรอย่าง ZoomText หรือ SuperNova คือหมวดหมู่ที่เหมาะสม แต่หากสิ่งที่คุณต้องการคือการซูมที่นุ่มนวลและสงบซึ่งติดตามเคอร์เซอร์ของคุณขณะทำงาน โดยไม่ต้องตั้งค่าใดๆ Low Vision Zoom ถูกสร้างขึ้นมาเพื่อกรณีนี้โดยเฉพาะ ในราคา $9.99 จ่ายครั้งเดียว เนื่องจากการขยายภาพเป็นเรื่องเฉพาะบุคคล จึงควรลองใช้ตัวเลือกใดก็ตามก่อนตัดสินใจจ่ายเงิน ดูการเปรียบเทียบอย่างตรงไปตรงมาของเรา

มีทางเลือกอื่นนอกจาก Windows Magnifier ไหม?

มี นอกเหนือจาก Windows Magnifier ที่ Microsoft ติดตั้งมาให้ในตัวแล้ว โปรแกรมขยายหน้าจอจากผู้พัฒนารายอื่นสำหรับ Windows ยังมี ZoomText และ SuperNova (ชุดโปรแกรมสำหรับผู้มีสายตาเลือนรางแบบครบวงจร มีทั้งเสียงพูดและเครื่องมือช่วยอ่าน) Magnifixer ที่ใช้งานได้ฟรี และ Low Vision Zoom ซึ่งเป็นโปรแกรมขนาดเล็กราคา $9.99 ที่เน้นการซูมแบบนุ่มนวลและยึดตามเคอร์เซอร์ ผู้คนมักมองหาทางเลือกอื่นเมื่อต้องการความรู้สึกที่นุ่มนวลกว่า การซูมที่ยึดตามตัวชี้เมาส์ หรือฟีเจอร์ด้านเสียงและสีที่เครื่องมือในตัวไม่มีให้ ทางเลือกใดจะเหมาะกับคุณขึ้นอยู่กับว่าคุณต้องการสิ่งใดในบรรดานี้ — หน้าเปรียบเทียบแสดงให้ดูเทียบกันแบบเคียงข้าง

มีทางเลือกที่ถูกกว่า ZoomText ไหม?

ขึ้นอยู่กับว่าคุณใช้ฟีเจอร์ใดของ ZoomText เป็นหลัก ZoomText เป็นชุดโปรแกรมสำหรับผู้มีสายตาเลือนรางแบบครบวงจร มีทั้งเสียงพูด เครื่องมืออ่านเอกสาร การปรับปรุงด้านสีและการโฟกัสอย่างละเอียด และการขยายภาพในระดับสูงมาก โดยทั่วไปจำหน่ายในรูปแบบใบอนุญาตรายปี หากคุณใช้ฟีเจอร์การอ่านและการปรับปรุงเหล่านั้น คำตอบที่ตรงไปตรงมาคือเครื่องมือที่ถูกกว่าจะไม่สามารถทดแทนได้ แต่หากสิ่งที่คุณต้องการหลักๆ คือการขยายภาพอย่างนุ่มนวลที่ติดตามเคอร์เซอร์ของคุณ Low Vision Zoom ทำสิ่งนั้นได้เพียงอย่างเดียวในราคา $9.99 จ่ายครั้งเดียว — แต่ไม่มีเสียงพูด ไม่มีการอ่าน ไม่มีตัวกรองสี และหยุดที่ 5× ส่วน Windows Magnifier ก็ใช้งานได้ฟรีเช่นกันและมีความสามารถมากกว่าที่หลายคนคิด

Low Vision Zoom อ่านข้อความออกเสียงหรือไม่?

ไม่ Low Vision Zoom ทำหน้าที่ขยายภาพเท่านั้น ไม่มีระบบแปลงข้อความเป็นเสียง ไม่มีเครื่องมืออ่านเอกสาร และไม่มีโปรแกรมอ่านหน้าจอ หากคุณต้องการให้อ่านหน้าจอออกเสียง ลองพิจารณา ZoomText Magnifier/Reader, SuperNova รุ่นที่มีเสียงพูดหรือโปรแกรมอ่านหน้าจอ หรือ Narrator ที่ Windows ติดตั้งมาให้ในตัวและใช้งานได้ฟรี

Low Vision Zoom แตกต่างจาก Windows Magnifier ที่ติดตั้งมาให้ในตัวอย่างไร?

Windows Magnifier ใช้งานได้ฟรี ติดตั้งมาให้ในตัว และมีความสามารถดี หลายคนจึงควรลองใช้ก่อนเป็นอันดับแรก Low Vision Zoom แตกต่างตรงความรู้สึกในการใช้งานมากกว่าจำนวนฟีเจอร์ การซูมของมันต่อเนื่องและค่อยๆ ผ่อนเข้าสู่ระดับใหม่อย่างนุ่มนวลแทนที่จะกระโดดเป็นขั้นๆ หมุนรอบจุดที่อยู่ใต้เคอร์เซอร์เมาส์ของคุณพอดี ทำให้สิ่งที่คุณกำลังมองอยู่ไม่ขยับ และมุมมองจะนิ่งอยู่กับที่ตราบใดที่เคอร์เซอร์ของคุณยังอยู่กลางจอ ตัวโปรแกรมไม่มีการตั้งค่าใดๆ และอยู่ในถาดระบบ นอกจากนี้ยังมีข้อจำกัดมากกว่า Windows Magnifier ในบางด้าน: ไม่มีเสียงพูด ไม่มีตัวกรองสี และหยุดที่ 5× อ่านเพิ่มเติมเกี่ยวกับเหตุผลเบื้องหลังการเคลื่อนไหวที่นุ่มนวลได้ที่เหตุใดการขยายภาพแบบนุ่มนวลจึงติดตามได้ง่ายกว่า

การใช้งาน Low Vision Zoom

ฉันจะซูมเข้าบน Windows 11 ได้อย่างไร?

Windows มีโปรแกรมขยายหน้าจอในตัว: กดปุ่ม Windows key ค้างไว้แล้วกด Plus เพื่อซูมเข้า กด Minus เพื่อซูมออก และกด Windows key + Esc เพื่อปิดการทำงาน คุณยังสามารถขยายทุกอย่างให้ใหญ่ขึ้นอย่างถาวรได้ที่ การตั้งค่า → ระบบ → การแสดงผล → มาตราส่วน หรือขยายเฉพาะข้อความได้ที่ การตั้งค่า → การช่วยการเข้าถึง → ขนาดข้อความ เมื่อติดตั้ง Low Vision Zoom แล้ว คุณสามารถกด Ctrl ค้างไว้แล้วเลื่อนล้อเมาส์เพื่อซูมได้อย่างนุ่มนวลทุกที่ใน Windows หรือใช้ Ctrl + Alt + Plus / Minus หากไม่มีล้อเมาส์

มีโปรแกรมขยายหน้าจอที่ซูมด้วยล้อเมาส์ไหม?

มี Low Vision Zoom ซูมด้วย Ctrl + ล้อเมาส์: กด Ctrl ค้างไว้แล้วหมุนล้อไปข้างหน้าเพื่อซูมเข้า หมุนกลับเพื่อซูมออก การซูมเป็นสัดส่วนโดยตรง คุณจะซูมได้ระยะและความเร็วตามที่คุณหมุนล้อพอดี และภาพจะค่อยๆ ผ่อนเข้าสู่ระดับใหม่อย่างนุ่มนวลโดยยังคงอยู่ตรงกลางที่จุดใต้ตัวชี้ของคุณ Ctrl + Shift + เลื่อนลง จะดีดกลับสู่ขนาดปกติทันที นี่คือวิธีหลักที่แอปนี้ถูกออกแบบมาให้ใช้งาน

Low Vision Zoom ใช้งานได้กับจอภาพหลายจอไหม?

ได้ โปรแกรมจะขยายเดสก์ท็อป Windows ทั้งหมดให้เป็นพื้นที่ต่อเนื่องเดียวกัน และติดตามเคอร์เซอร์ของคุณจากจอหนึ่งไปยังอีกจอหนึ่ง แทนที่จะแยกจัดการแต่ละจอต่างหาก นี่เป็นแนวทางเดียวกับที่โหมดเต็มจอของ Windows Magnifier ที่ติดตั้งมาให้ในตัวใช้

ฉันจะซูมบนแล็ปท็อปที่ไม่มีล้อเมาส์ได้อย่างไร?

ใช้แป้นพิมพ์ กด Ctrl + Alt ค้างไว้แล้วกด Plus เพื่อซูมเข้า หรือ Minus เพื่อซูมออก กดปุ่มค้างไว้เพื่อซูมต่อเนื่อง Ctrl + Alt + 0 จะรีเซ็ตกลับสู่ขนาดปกติ ไม่จำเป็นต้องมีล้อเมาส์หรือเมาส์ภายนอก แอปจึงใช้งานได้บนแล็ปท็อปที่มีแค่ทัชแพด

ใช้งานได้ในเกมและวิดีโอเต็มจอไหม?

ใช้งานได้เกือบทุกที่ Low Vision Zoom ขยายเดสก์ท็อป Windows ทั้งหมด จึงครอบคลุมเบราว์เซอร์ อีเมล เอกสาร วิดีโอที่เล่นในหน้าต่าง และเชลล์ของ Windows เอง ข้อยกเว้นคือเกมหรือแอปที่เข้าควบคุมหน้าจอแบบเอ็กซ์คลูซีฟทั้งหมดและข้ามเดสก์ท็อปปกติไป เนื้อหานั้นอาจไม่ถูกขยายภาพ การเรียกใช้เกมดังกล่าวในโหมดหน้าต่างหรือโหมดหน้าต่างไร้ขอบจะทำให้การขยายภาพกลับมาทำงานตามปกติ

Low Vision Zoom สามารถขยายหน้าจอได้มากแค่ไหน?

สูงสุด 5× เป็นขั้นละครึ่งเท่า นี่เป็นข้อจำกัดที่ตั้งใจไว้สำหรับเครื่องมือซูมขนาดเล็กที่ใช้งานสบาย และต่ำกว่า Windows Magnifier ที่ติดตั้งมาให้ในตัว (สูงสุด 16×) หรือชุดโปรแกรมครบวงจรอย่าง ZoomText (สูงสุด 36×) หากคุณต้องการมากกว่า 5× Low Vision Zoom ไม่ใช่เครื่องมือที่เหมาะกับคุณ — หน้าเปรียบเทียบจะแนะนำตัวเลือกที่ขยายได้สูงกว่านี้

การซื้อ การทดลองใช้ และความน่าเชื่อถือ

Low Vision Zoom ราคาเท่าไหร่?

เป็นใบอนุญาตแบบจ่ายครั้งเดียวราคา $9.99 รวมการอัปเดตในอนาคตทั้งหมด ไม่มีค่าสมัครสมาชิก ไม่ต้องสร้างบัญชี และไม่มีโฆษณา คุณสามารถทดลองใช้ทุกฟีเจอร์ฟรี 7 วันก่อน และมีการรับประกันคืนเงินภายใน 30 วันหลังการซื้อ ดูหน้าราคา

จะเกิดอะไรขึ้นเมื่อช่วงทดลองใช้ฟรีสิ้นสุดลง?

แอปยังคงทำงานต่อไป เมื่อช่วงทดลองใช้ 7 วันสิ้นสุดลง การขยายภาพจะถูกจำกัดเท่านั้น ไม่ใช่ปิดการทำงาน ทำให้หน้าจอของคุณยังคงอ่านได้เพียงพอสำหรับการใช้งานทั่วไปและการซื้อใบอนุญาต กรอกคีย์ใบอนุญาตเมื่อใดก็ได้เพื่อคืนการซูมเต็มรูปแบบทันที โดยไม่ต้องติดตั้งใหม่

Low Vision Zoom เก็บข้อมูลใดๆ หรือไม่ และใช้งานแบบออฟไลน์ได้ไหม?

แอปนี้ไม่เก็บข้อมูลส่วนบุคคล ไม่มีการวิเคราะห์การใช้งาน และไม่มีการส่งข้อมูลทางไกล (telemetry) และทำงานได้แบบออฟไลน์เต็มรูปแบบ แอปไม่บันทึกหรือส่งเนื้อหาบนหน้าจอของคุณออกไปที่ใด คีย์ใบอนุญาตของคุณจะถูกตรวจสอบบนอุปกรณ์ของคุณเอง โดยไม่ต้องมีบัญชีและไม่มีการเชื่อมต่อกลับเซิร์ฟเวอร์อย่างเงียบๆ ช่วงเวลาเดียวที่แอปเชื่อมต่ออินเทอร์เน็ตคือเมื่อคุณเลือก Check for updates (ตรวจสอบการอัปเดต) จากเมนูถาดระบบ รายละเอียดทั้งหมดอยู่ในนโยบายความเป็นส่วนตัว (ภาษาอังกฤษ)

Windows จะเตือนฉันตอนติดตั้ง Low Vision Zoom ไหม?

อาจจะใช่ หากคุณใช้การดาวน์โหลดโดยตรง ครั้งแรกที่คุณเปิดตัวติดตั้ง Windows อาจแสดงข้อความ SmartScreen สีน้ำเงินว่า “Windows protected your PC” (Windows ปกป้อง PC ของคุณ) นี่เป็นเรื่องปกติสำหรับแอปใหม่และปลอดภัย ตัวติดตั้งมีลายเซ็นดิจิทัลจาก txtbook LLC คลิก More info (ข้อมูลเพิ่มเติม) ตรวจสอบว่าผู้เผยแพร่คือ txtbook LLC แล้วคลิก Run anyway (เรียกใช้ต่อไป) หากต้องการข้ามหน้าจอนี้ไปเลย ให้ติดตั้งจาก Microsoft Store แทน ซึ่งจะไม่แสดงคำเตือนนี้

ข้อกำหนดระบบมีอะไรบ้าง?

Low Vision Zoom ทำงานบน Windows 10 และ Windows 11, 64 บิต (x64) ตัวติดตั้งครบในตัว จึงไม่ต้องดาวน์โหลดอย่างอื่นเพิ่มเติม ยังไม่รองรับ Windows on ARM และพีซี 32 บิตในขณะนี้
